# ShitSalary - Монетки за уничтожение мобов

### [-] Зависимости:
* Paper версии 1.18+
* Vault
* Любой плагин на экономику (Essentials, CMI, другое)

### [-] Пример конфигурации
```yaml
entities:
  player: # id энтити
    price: # Сколько монеток даёт данный энтити
      min: 100 # можно указать дробью. 100.5 | 100.52131 | 0.1488
      max: 200
    type-withdrawal: virtual # virtual | physical Должен ли выпадать физический предмет монетки
```
